◆ cbmimage_i_fmt_print()

void cbmimage_i_fmt_print ( const char * fmt,
... )

send formatted output from the library

Parameters
[in]fmtpointer to a printf() format string that is output.
[in]...additional parameter needed for the fmt format string
Remarks
  • As (vsn)printf() is used internally, all formatting allowed by printf() can be used, too.
  • If a string without printf() style formatting shall be sent, use the more efficient cbmimage_i_print() instead.
